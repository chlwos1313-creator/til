# [String] [D4] 여우 줄이기

## 1. 문제 개요
- **문제:** 길이 $N$의 문자열에서 "fox"라는 부분 문자열을 반복적으로 지워나갈 때, 남는 문자열의 최소 길이를 구하시오. (지워진 후 다시 합쳐져서 "fox"가 되는 경우도 처리 필요)
- **입력:**
    - 첫 줄: 테스트 케이스 수 $TC$
    - 각 케이스: 첫 줄에 길이 $N$, 두 번째 줄에 문자열
- **출력:** 각 테스트 케이스마다 남은 문자열의 길이
- **유형:** 자료구조 (Stack), 문자열 처리

## 2. 접근 방법 (Why Stack?)
- **문제점:**
    - `replace()` 함수를 반복문(`while`)으로 계속 돌리면, 문자열이 길어질 때마다 매번 새로 생성해야 함.
    - 이로 인해 연산량이 급격히 늘어나 **시간 초과(Time Limit)**가 발생할 위험이 있음.
- **해결책:**
    - **스택(Stack)**을 사용하여 문자열을 한 번만 훑으면서($O(N)$) 처리함.
    1. 문자를 하나씩 **바구니(Stack)**에 담는다.
    2. 담을 때마다 바구니의 **끝부분 3글자**가 `fox`인지 확인
    3. 맞다면 즉시 제거(`del`)

## 3. 핵심 문법 정리 

### ① 슬라이싱 (Slicing)
- **`bag[-3:]`**: 리스트의 뒤에서 3번째부터 끝까지. 
- **`bag[::-1]`**: 리스트 뒤집기. 

### ② 삭제 연산
- **`bag.pop()`**: 맨 뒤의 요소를 꺼내면서 삭제.
- **`del bag[-3:]`**: 해당 범위의 요소를 **한 번에 삭제**. (메모리 효율 및 코드 간소화)

### ③ 입력 처리
- **`input()`**: 무조건 문자열(String)로 받음.
- **`int(input())`**: 숫자로 계산할 경우 반드시 형변환 필요.

## 4. 최종 코드 (Python)

```python
import sys
# sys.stdin = open("input.txt", "r")

T = int(input())

for test_case in range(1, T + 1):
    # 1. 길이 N 입력 (숫자로 변환)
    N = int(input())
    
    # 2. 문자열 입력
    text = input()
    
    # 3. 빈 바구니(Stack) 준비
    bag = []
    
    for char in text:
        # 글자를 하나씩 바구니에 담기
        bag.append(char)
        
        # 4. 바구니에 3글자 이상 있고, 끝이 'fox' 인지 확인
        # bag[-3:] -> 뒤에서부터 3개를 싹 긁어옴
        if len(bag) >= 3 and bag[-3:] == ['f', 'o', 'x']:
            
            # 5. fox가 맞다면 통째로 삭제 (Slicing Delete)
            # pop() 3번 대신 del을 사용하여 간결하게 처리
            del bag[-3:]
            
    # 6. 남은 문자열의 길이 출력
    print(len(bag))
