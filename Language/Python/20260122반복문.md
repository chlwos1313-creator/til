## 반복문 (Iteration)
반복문의 정의 : 동일한 코드 블록을 특정 조건이 만족될 때까지 반복적으로 실행하는 구조

### for 문
- **특징**: 반복 가능한 객체(Iterable)의 요소들을 순차적으로 순회
- **구조**: `for 변수 in 반복 가능한 객체:`

**for 문 예제**
```python
fruits = ['apple', 'banana', 'cherry']
for fruit in fruits:
    print(fruit)

# 시작 값, 끝 값, 증감 폭 지정 가능
for i in range(1, 10, 2):
    print(i) # 1, 3, 5, 7, 9 출력
    while 문
```
특징: **조건식이 참(True)** 인 동안 계속해서 로직을 실행

주의사항: 루프 내에서 조건식을 변경하지 않으면 무한 루프에 빠질 수 있으므로 주의해야 함

while 문 예제

```python
count = 0
while count < 3:
    print(f'현재 횟수: {count}')
    count += 1
```
3. **반복 제어 (Control Flow)**
break: 반복문을 즉시 종료하고 탈출

continue: 현재 반복의 남은 로직을 건너뛰고 다음 반복으로 진행

for-else: 반복문이 break를 만나지 않고 끝까지 정상 실행되었을 때만 else 블록을 실행

리스트 컴프리헨션 (List Comprehension)
컴프리헨션 정의
기존의 반복 가능한 객체를 기반으로 새로운 리스트를 생성하는 간결하고 효율적인 문법

컴프리헨션 예제

```Python
# 1~10 중 짝수만 추출하여 새로운 리스트 생성
evens = [x for x in range(1, 11) if x % 2 == 0]
```
반복문 스타일 가이드 및 원칙

변수명: 소문자와 언더스코어를 사용하며, 반복되는 대상을 명확히 설명해야 함
중첩 방지: 이중 루프 이상의 중첩은 알고리즘의 효율성(시간 복잡도)을 저하시키므로 지양함
